Справка по обработке БонуснаяСистема

Обработка предназначена для управления бонусной программой, включает начисление, списание и настройку правил расчёта бонусов. Взаимодействует с документами реализации, справочниками и регистрами накопления.

Форма ФормаНачисленияБонусов

Назначение:
Используется для начисления бонусов по документу. Вызывается из документа при нажатии кнопки «Начислить», если по документу ещё не производилось начисление бонусов.

Основной функционал:

  • Отображение суммы бонусов к начислению.

  • Подтверждение начисления через СМС-код.

  • После подтверждения вызывается серверный метод НачислитьБонусы, производящий движение Приход в регистре БонусыПокупателей.

Особенности:

  • При успешном начислении форма закрывается с выводом сообщения пользователю.

  • Повторное начисление по одному и тому же документу невозможно.

Форма ФормаСписанияБонусов

Назначение:
Используется для списания бонусов при оформлении продажи. Вызывается из документа при нажатии кнопки «Списать», если по документу ещё не было списания.

Основной функционал:

  • Ввод суммы бонусов, подлежащих списанию.

  • Подтверждение списания через СМС-код.

  • После подтверждения вызывается серверный метод СписатьБонусы, производящий движение Расход в регистре БонусыПокупателей.

Особенности:

  • В случае успешного списания формируется возврат результата, чтобы сумма скидки могла быть применена в документе.

  • Повторное списание невозможно — предусмотрена проверка.

Форма ФормаНастроекБонусов

Назначение:
Предоставляет доступ к настройке правил расчёта бонусов и условий участия покупателей в бонусной программе.

Основной функционал:

  • Управление видами цен, участвующих в расчёте бонусов.

  • Задание индивидуальных настроек для покупателей.

  • Возможность настройки коэффициентов.

Особенности:

  • Открывается как независимая форма (не блокирует владельца).

  • Используется для предварительной настройки перед запуском бонусной системы в эксплуатацию.
